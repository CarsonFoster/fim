fn setup(&mut self, _ed: &mut Editor<'_>) -> Result<bool>

Function to setup the Context when it becomes the active context.

Can cause the Context to ‘return’ and be popped off the context stack, if this function returns Ok(true). If this function returns Ok(false), the Context has not ‘returned’.
